# Ambivare - Advanced Coaching Institute Management System

A comprehensive web-based CRM system designed specifically for coaching institutes to manage students, teachers, classes, attendance, and academic activities.

## 🌟 Features

### Current Features
- **Admin Portal**: Complete user management, class management, task assignment, and analytics dashboard
- **Teacher Portal**: OTP-based attendance system, homework management, task tracking, and class management
- **Student Portal**: OTP attendance marking, personalized dashboard, homework tracking, and timetable access
- **Real-time Data**: All data is synchronized across portals in real-time
- **Mobile Responsive**: Fully responsive design that works on all devices

### Upcoming Features
- Advanced analytics and reporting
- Email notifications
- Mobile applications
- Video integration
- Payment management
- Parent portal

## 🚀 Quick Start

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- No server setup required - runs entirely in the browser

### Installation
1. Clone or download the repository
2. Open `index.html` in your web browser
3. Start using the system!

## 🚀 Deployment

### Netlify Deployment
1. Connect your GitHub repository to Netlify
2. Set build settings:
   - Build command: `echo "Static site - no build required"`
   - Publish directory: `.`
3. Deploy!

### Custom Domain
- Point your domain `ambivare.com` to Netlify
- SSL certificate will be automatically provisioned
